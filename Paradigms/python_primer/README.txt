This program uses movie data (in /ml-1m) to build a database.
_movie_database.py loads movies, ratings, and users information into dictionaries.
With the movies dictionary, the user can load, get, set, and delete movie information.
With the users dictionary, the user can load, set, delete user information. 
  The user can also get individual user information or the whole dictionary.
With the ratings dictionary, the user can load, get, and delete rating information.
  The user can also get the highest rated movie and get or set a user's rating of a movie.

test_movie_database.py tests the functionality of the _movie_database.py script.
